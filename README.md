# Moon Store

متجر إلكتروني متكامل لملابس وملحقات التخرج الفاخرة — React + Vite +
Tailwind من جهة الواجهة، و [Supabase](https://supabase.com) (قاعدة بيانات +
مصادقة + تخزين ملفات) من جهة الباك اند. مستقل بالكامل، بدون أي اعتماد على
منصات خارجية مغلقة.

## البداية السريعة

راجع [`SETUP.md`](./SETUP.md) لخطوات ربط المشروع بـ Supabase وتشغيله محليًا
(حوالي 15 دقيقة أول مرة).

```bash
npm install
npm run dev
```

## البنية

```
src/
  api/            # طبقة الوصول للبيانات (entities, auth, storage, email)
  components/     # مكوّنات واجهة قابلة لإعادة الاستخدام
  components/admin/  # مكوّنات لوحة التحكم
  components/sections/  # أقسام الصفحة الرئيسية
  lib/            # سياقات React (Auth, Cart, Settings) وأدوات مساعدة
  pages/          # صفحات الموقع (كل صفحة = مسار)
supabase/
  schema.sql          # كل جداول قاعدة البيانات + الصلاحيات (RLS)
  functions/send-email # Edge Function اختيارية لإرسال البريد
```

## المزايا

- عرض المنتجات، تصفية حسب الفئة، بحث
- سلة مشتريات وصفحة دفع كاملة (بطاقة / الدفع عند الاستلام) مع أكواد خصم
- تتبع الطلبات (بدون تسجيل دخول) عبر رقم الطلب أو الجوال
- تسجيل دخول/تسجيل حساب (بريد وكلمة مرور، أو Google)
- لوحة تحكم كاملة: الطلبات، المنتجات، العملاء، الأرباح، الموردين، أكواد
  الخصم، السلات المتروكة، إعدادات المتجر
- محاكي تصميم الوشاح (Sash Simulator)

## الأوامر

| الأمر | الوصف |
|---|---|
| `npm run dev` | تشغيل محلي |
| `npm run build` | بناء نسخة الإنتاج (`dist/`) |
| `npm run preview` | معاينة نسخة الإنتاج محليًا |
| `npm run lint` | فحص الكود |
